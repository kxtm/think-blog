Think-Blog
===============

## 环境

> ThinkPHP8的运行环境要求PHP8.0+

## 框架文档

[完全开发手册](https://doc.thinkphp.cn)

## 安装

~~~
composer create-project topthink/think tp
~~~

启动服务

~~~
cd tp
php think run
~~~

然后就可以在浏览器中访问

~~~
http://localhost:8000
~~~

如果需要更新框架使用
~~~
composer update topthink/framework
~~~

## 命名规范

`ThinkPHP`遵循PSR-2命名规范和PSR-4自动加载规范。
